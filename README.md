# Geladaria & Hamburgeria KENDIS

Plataforma digital integrada para vendas online, delivery, reservas e fidelização.

## Estrutura do Projeto

- **frontend/**: Aplicação Next.js (Web e Mobile-first).
- **backend/**: API RESTful Java com Spring Boot.
- **docs/**: Documentação do projeto.

## Tecnologias

- **Frontend**: React, Next.js 14+, Tailwind CSS, TypeScript.
- **Backend**: Java 17+, Spring Boot 3.x, PostgreSQL.
- **Autenticação**: Spring Security + JWT / Supabase Auth.
- **Banco de Dados**: PostgreSQL (Supabase).

## Como Rodar

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
Necessário ter JDK 17+ e Maven instalados.
```bash
cd backend
mvn spring-boot:run
```
